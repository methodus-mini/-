# 🍅 온라인 뽀모도로 타이머 (Pomodoro Timer)

집중력 향상을 위한 가장 간단하고 강력한 방법, **뽀모도로 기법**을 실천할 수 있는 온라인 타이머입니다.

## 🚀 서비스 바로가기
[(https://pomodoro-mini-timer.netlify.app/)]
## ✨ 주요 기능
- **25분 집중 / 5분 휴식:** 가장 효율적인 뽀모도로 사이클을 기본으로 제공합니다.
- **깔끔한 UI:** 광고나 복잡한 설정 없이 타이머에만 집중할 수 있는 미니멀한 디자인.
- **반응형 디자인:** PC는 물론 모바일 브라우저에서도 최적화된 화면을 제공합니다.

## 🛠 사용 기술
- **Frontend:** HTML5, CSS3, JavaScript
- **Deployment:** Netlify
- **Version Control:** GitHub

## 📖 사용 방법
1. 사이트에 접속합니다.
2. 'Start' 버튼을 눌러 25분 집중을 시작합니다.
3. 알람이 울리면 5분간 휴식을 취합니다.
4. 이 과정을 반복하여 업무나 공부의 효율을 높이세요!
